Proyectos Destacados

Lytics

Lytics es una aplicación diseñada para ayudar a los usuarios domésticos a monitorear, gestionar y optimizar su consumo energético de manera efectiva. Este proyecto combina tecnología avanzada con una interfaz amigable para empoderar a los usuarios en la toma de decisiones que reduzcan su huella de carbono y sus costos energéticos. Características principales: Monitoreo en tiempo real: Ofrece datos actualizados sobre el consumo energético de diferentes dispositivos en el hogar, permitiendo identificar patrones de uso y áreas de mejora. Pagos integrados con Mercado Pago: Facilita la gestión financiera al permitir a los usuarios realizar pagos de servicios directamente desde la aplicación, de forma rápida y segura. Sección de tutoriales interactivos: Proporciona guías prácticas y fáciles de seguir para optimizar el uso de energía, adoptar prácticas más sostenibles y reducir el desperdicio energético. Huella de carbono: Calcula y presenta la huella de carbono generada por el consumo del usuario, motivando a implementar hábitos más ecológicos. Notificaciones inteligentes: Envía alertas y recomendaciones personalizadas según el consumo del usuario y las tarifas actuales de energía. Tecnologías utilizadas: Frontend: Angular, Tailwind CSS Backend: Java Spring Boot Pagos: Integración con la API de Mercado Pago Visualización de datos: Gráficos y estadísticas generados dinámicamente para facilitar el análisis del consumo. Impacto del proyecto: Lytics busca no solo simplificar la gestión del consumo energético, sino también educar a los usuarios sobre la importancia del ahorro de energía y la sostenibilidad. Esta herramienta es ideal para hogares que desean modernizar su gestión eléctrica y reducir sus costos de manera consciente y sostenible.
